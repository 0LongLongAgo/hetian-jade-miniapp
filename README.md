# 💎 独作 — 和田玉小程序

基于微信小程序的玉石鉴定与手串定制平台。

## ✨ 功能

- **玉石百科** — 和田玉材质分类、鉴别知识
- **手串定制** — 在线选珠、搭配、设计手串
- **3D 预览** — 手串 3D 效果展示
- **设计画廊** — 浏览其他用户的设计作品
- **社区广场** — 分享和交流
- **玉石识别** — 拍照识别玉石材质
- **后台管理** — 管理员内容管理系统

## 🚀 使用

1. 安装[微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 导入本项目文件夹
3. 填写自己的 AppID（`project.config.json`）
4. 预览/上传

## 🔧 技术栈

- 微信小程序原生框架
- JavaScript + WXML + WXSS
